# 🏠 House Price Prediction Web Application

A Machine Learning–powered web application that predicts residential property prices based on key housing attributes. The solution integrates data preprocessing, model training, evaluation, and deployment through a Flask-based user interface.

---

## Project Overview

This project applies supervised machine learning techniques to estimate house prices using features such as overall quality, living area, garage information, basement size, construction year, bathrooms, and room count.

After preprocessing and feature selection, multiple regression models were evaluated and the selected model was deployed as an interactive Flask web application for real-time predictions.

---

## Key Features

✔ Data cleaning and preprocessing  
✔ Feature engineering and selection  
✔ Machine learning model training and evaluation  
✔ Model serialization using Joblib (`.pkl`)  
✔ Flask web application deployment  
✔ Interactive and responsive user interface
